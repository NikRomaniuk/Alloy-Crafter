## Quick Start

### Install Dependencies
Ensure you have [Node.js](https://nodejs.org/en) installed, then run:
   ```bash
   npm install
   ```

### Project Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/NikRomaniuk/Alloy-Crafter.git
   cd alloy-crafter
   ```

2. **Launch the Application**
   ```bash
   ionic serve
   ```
